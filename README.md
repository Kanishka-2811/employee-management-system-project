# Employee Management System

A full-stack web application designed to manage employee records efficiently. Built with a robust backend and an interactive frontend interface.

## 🚀 Tech Stack

- **Frontend:** HTML, CSS, JavaScript (React / Vanilla JS)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
